GeoNEX-core Restful APIs Document

  1. Start Runner

  2. Name: Start RealTime Runner

  3. Description: This api is used to start a real-timer runner in GEONEX dash
  4. Http Request Method: GET
  5. URL: /start-runner
  6. Input Parameter:

instance_type : String image_name: String s3_output: String runner_id : String runner_name : String

  • Output Parameter: success or error code

  • Request Example: /start-runner?instance_type=m5d.4xlarge&image_name=geonex/active_fire_go16abi15:1.5&s3_output=s3://geonex/GO16_ABI15&runner_id=8dcdfcd4-c21a-c8ba-5152-9475dea5f1bd&runner_name=Production
